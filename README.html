Zen Tetris

Zen Tetris is yet another Tetris clone written in Python. Sounds uninteresting? I understand…but wait! This is not an ordinary Tetris clone! So, what’s different about Zen Tetris?

  • Cool, Zen-feeling opening view (Look at the drifting Buddha statue!)
  • Good old 1-player mode with a stunning, rotating background.
  • Exciting two-player battle mode (Yey!)

Um. Two of the three above items are thanks to beautiful images I borrowed from someone on the net. And by the way, this is my first Python program. I hope you’ll enjoy the game.

How to run game

  • Download tet.py, images and sounds folder in the same folder
  • pip3 install arcade –user
  • python tet.py

How to play

Zen Tetris has two modes. In 1-player mode, you can play a good, old Tetris In 2-player battle mode, two players can “battle” with Tetris. If one player deletes more than one lines, (deleted lines - 1) lines will be added to the other player’s play area. Wow, sounds exciting!

Title/gameover view

  • Push “O” key to play 1-player game
  • Push “T” key to play 2-player battle game

Game view

  • Space bar - pause/cancel
  • ESC key - quit game

player 1 * Left arrow or “A” key - move left * Right arrow or “D” key - move right * Up arrow or “W” key - rotate * Down arrow or “S” key - drop

player 2 * “J”/“L” key - move left/right for 2nd player * “I” key - rotate for 2nd player * “K” key - drop for 2nd player
